1-1.  프로그램 구성 및 설치 전 준비 사항
이 프로그램은 DeepL API를 사용하여 고품질의 번역을 제공하며, 화면의 텍스트를 인식하기 위해 Tesseract OCR 엔진을 사용합니다.

Cross-reader.exe  프로그램을 실행하는 파일입니다.
config.py		설정(API 키, 경로 등)을 관리하는 파일입니다. (위 exe파일과 같은 폴더에 있어야 합니다.)

1-2. 필수 준비 사항 (사용자 PC)
Tesseract OCR 엔진 설치:

[Tesseract OCR 공식 다운로드 페이지]에서 Windows 설치 파일을 다운로드하여 설치해야 합니다.

-Tesseract 언어 파일 설치 (한글/일본어 등):

Tesseract 설치 시, 사용하려는 언어 팩을 선택하여 함께 설치하거나, 또는 아래 경로에 직접 .traineddata 파일을 복사해야 합니다.

경로 예시: C:\Program Files\Tesseract-OCR\tessdata

예시: 한국어 번역을 원하면 kor.traineddata 파일을 이 폴더에 넣어주세요.

-DeepL API 키 발급:

DeepL Pro 또는 DeepL API Free 계정에서 API 키를 발급받아야 합니다.

2.  프로그램 초기 설정
프로그램을 처음 실행하면, 캡처/번역을 시작하기 전에 반드시 다음 두 가지 정보를 입력해야 합니다.

2.1. 엔진/API 설정
프로그램 메인 화면의 " 엔진/API 설정" 섹션으로 이동합니다.

Tesseract 경로 (.exe): 설치된 Tesseract의 실행 파일 경로를 입력하거나, '찾아보기' 버튼을 클릭하여 tesseract.exe 파일의 위치를 지정합니다.

일반적인 경로: C:\Program Files\Tesseract-OCR\tesseract.exe

DeepL API 키: 발급받은 DeepL API 키를 입력합니다.

'설정 저장' 버튼을 눌러 모든 설정을 저장합니다. (설정 정보는 config.json 파일에 저장됩니다.)

2.2. 번역 언어 설정
"번역 컨트롤" 섹션에서 번역의 입/출력 언어를 지정합니다.
설정 항목,입력 값 예시,설명
대상 언어 코드 (DeepL),"KO (한국어), EN-US (미국식 영어)",번역 결과가 나타나길 원하는 언어입니다.
원본 언어 코드 (Tesseract OCR),"eng (영어), kor (한국어)",캡처하려는 원본 텍스트의 언어입니다. 이 설정이 잘못되면 OCR 인식이 깨집니다.

3. 번역 실행 방법
단축키 설정: **"3. 단축키 설정"**에서 원하는 단축키(기본값: ctrl+alt+t)를 확인하거나 '단축키 입력' 버튼을 클릭하여 새로 설정합니다.

감지 시작: '설정 적용 및 감지 시작' 버튼을 클릭합니다. 버튼이 **'감지 중지'**로 바뀌고 하단 상태 표시줄이 **'단축키 감지 중'**으로 변하면 준비 완료입니다.

캡처 실행: 설정한 단축키를 누릅니다.

영역 선택 모드: 화면이 어두워지면 마우스로 원하는 영역을 드래그하여 캡처합니다.

결과 확인: 잠시 후 '번역 결과 상세' 창이 나타나 번역 결과와 오버레이 뷰를 보여줍니다.

4. 오류

문제,원인 및 해결책

1. OCR 인식이 깨져 나옴,1순위: '원본 언어 코드 (Tesseract OCR)' 설정이 캡처 화면의 언어와 맞지 않습니다. (예: kor 대신 eng로 설정됨) 2순위: tessdata 폴더에 해당 언어 파일(.traineddata)이 없습니다.

2. Tesseract 오류 (파일 로드 실패),"Tesseract 설치 폴더에 해당 언어 파일(kor.traineddata 등)이 없거나, Tesseract 경로가 잘못 설정되었습니다."

3. .DeepL API 오류,"DeepL API 키가 잘못 입력되었거나, API 사용량이 초과되었습니다."
